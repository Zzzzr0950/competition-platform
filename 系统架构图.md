## 方案二：简易画法（在 WPS/PPT 里用矩形框画）

```
浏览器/手机
     │
     ▼
[ 前端 HTML+CSS+JS ]
     │  HTTP 请求
     ▼
[ Flask Web 服务器 ]
     │
     ├── auth.py     (登录/注册/改密)
     ├── student.py  (申报/查看)
     ├── admin.py    (审核/统计/导出)
     ├── upload.py   (文件上传)
     └── config.py   (配置/密钥)
     │
     ▼
[ SQLite 数据库 ]
     │
     ├── users              (用户表, 1257条)
     ├── submissions        (申报表)
     ├── competition_catalog(竞赛目录, 125条)
     └── submission_logs    (操作日志)
     │
     ▼
[ PythonAnywhere 云平台 ]
     │
     └── GitHub 自动部署
```
